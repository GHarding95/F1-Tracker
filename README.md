# F1-Tracker

Single page application for tracking current F1 Driver Standings using the Ergast F1 API: http://ergast.com/api/f1/current/driverstandings.json

f1-tracker-zeta.vercel.app/

## Tech Stack

- VS Code
- React JS
- TypeScript
- Tailwind CSS
- React-Router-DOM

## Getting Started

npm install

## To Run

npm start
